Proyecto Fin de Grado para FPGS de Desarrollo de Aplicaciones Multiplataforma (promoción '24-'26), desarrollado por Lucas Masa Pulido para AndroidTV

Homegym es una aplicación que espera traer el ambiente de un gimnasio a casa mediante el uso de televisiones inteligentes con el sistema operativo Android 11 o superior
El hosting del backend se realiza en un dispositivo local mientras que el frontend se puede cargar desde el emulador para AndroidTV de Android Studio
La aplicación presenta los aspectos básicos de un CRUD: crear usuarios, ejercicios, rutinas, leerlos desde la base de datos (operada con MySQL), actualizarlos o borrarlos. Todo esto se puede visualizar mediante Swagger para un fácil acceso a los datos desde localhost:8081/swagger
Dentro de la aplicación se encuentra una pantalla inicial donde iniciar sesión, registrarse o acceder como invitado. Una vez pasado a la pantalla principal, se puede acceder a diferentes secciones mediante los diferentes scroll-to (para los vídeos), o los buttons (ajustes, log out, FAQs, pestañas de rutinas e historial...)
